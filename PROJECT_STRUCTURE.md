# Project Structure

This document describes the main files and directories in this repository.

## Root Directory

- **README.md**: main project documentation with context, setup, and execution examples.
- **PROJECT_STRUCTURE.md**: reference file describing the repository layout.
- **main.py**: simple entry point for local execution.
- **e01_age_checker.py**: checks age-based conditions.
- **e02_discount_calculator.py**: applies discount rules based on scenario conditions.
- **e03_ticket_triage.py**: classifies support tickets according to defined criteria.
- **e04_parity_checker.py**: verifies whether a number is even or odd.
- **e05_compare_two_numbers.py**: compares two values and reports the result.
- **e06_login_access.py**: simulates a basic login or access validation flow.
- **e07_bmi_calculator.py**: calculates BMI and classifies the result.
- **e08_grade_classifier.py**: evaluates grades according to approval rules.
- **e09_salary_adjustment.py**: calculates salary changes using predefined conditions.
- **e10_triangle_analyzer.py**: validates and classifies triangle types.
- **e11_simple_calculator.py**: performs basic arithmetic operations.
- **e12_unit_converter.py**: converts measurement units.
- **e13_atm_simulator.py**: simulates ATM withdrawal logic.
- **pyproject.toml**: Python project configuration.
- **uv.lock**: dependency lock file generated by `uv`.
- **.python-version**: defines the Python version used locally.
- **.gitignore**: lists files and directories ignored by Git.

## Directories

- **docs/**: supporting documentation, mentoring references, and project notes.
- **tests/**: lightweight test files used to validate selected exercises.
- **.venv/**: local virtual environment.
- **.git/**: internal Git repository data.

## Organization Notes

This repository follows a simple rule: exercise files stay in the root when they are part of the same list and benefit from quick access. Supporting material stays separated in `docs/` and `tests/` to keep the project readable.
